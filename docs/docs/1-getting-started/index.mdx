# Getting Started {class="text-2xl font-semibold mt-4 !mb-4"}

The superheroic UI collection of front-end libraries.

## About {class="text-xl font-semibold !mb-4"}

**This** is a UI toolkit for developing with HTML, CSS, and JS. Built on top of TailwindCSS and Bootstrap.
Build responsive, mobile-first projects on the web with this front-end component library. Quickly
prototype your ideas or build your entire app with responsive grid system, extensive prebuilt components,
and powerful plugins built on jQuery.

## CDN {class="text-xl font-semibold !mb-4"}

Many of our components require the use of JavaScript to function.
Place the following `<script>`s near the end of your pages, right before the closing`</body>` tag, to enable them.
